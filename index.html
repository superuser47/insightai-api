Try It Now

curl -X POST https://api.insightai.dev/api/v1/analyze \
  -H "Content-Type: application/json" \
  -d '{
    "data": [
      {"value": 10, "label": "day1", "category": "sales"},
      {"value": 15, "label": "day2", "category": "sales"},
      {"value": 100, "label": "day3", "category": "sales"}
    ],
    "domain": "business"
  }'

# Response:
{
  "insights": [
    {"type": "anomaly", "description": "Day 3 shows unusual spike (z=2.95)", "confidence": 0.95}
  ],
  "processing_time_ms": 0.8
}
